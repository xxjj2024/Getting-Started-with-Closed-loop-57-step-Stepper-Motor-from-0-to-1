# Getting-Started-with-Closed-loop-57-step-Stepper-Motor-from-0-to-1
This article mainly uses the stm32f103c8t6 to gradually teach beginners how to quickly get started with the closed-loop 57-step motor. It starts with the standard library and gradually iterates to become industrial-grade code under the standard library, making it relatively friendly for beginners.
