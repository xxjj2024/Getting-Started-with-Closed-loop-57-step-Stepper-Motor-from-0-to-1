在引入了 TTL 串口通信后，你的系统看似很完美，但实际上多了一个极其致命的弱点：**通信线缆脱落或上位机死机**。

想象一下：你给传送带发了一个大写字母 `'A'`（全速前进），电机开始狂奔。就在这时，你不小心把杜邦线踢断了，或者电脑蓝屏了！ 你的 STM32 永远收不到刹车指令 `'S'`，传送带就会像脱缰的野马一样一直冲下去，直到把机械结构全部撞毁。

为了填补这个终极漏洞，我们现在就把**独立看门狗 (IWDG, Independent Watchdog)** 请出来！

---

### 第一层透视：看门狗的物理真相（为什么叫“独立”？）

STM32 内部其实有两条狗：一只叫窗口看门狗 (WWDG)，一只叫独立看门狗 (IWDG)。在防死机和防断线的场景中，我们只用 **IWDG**。

- **真正的物理隔离：** 为什么叫“独立”？因为这只狗极其有骨气，它**根本不用 CPU 的 72MHz 高速时钟**。它有自己专属的、直接刻在硅片上的“低速内部晶振 (LSI, 大约 40kHz)”。
    
- **大白话：** 哪怕你的 STM32 主频晶振被雷劈坏了，CPU 彻底死机瘫痪了，这只狗依然在默默倒计时！一旦倒计时结束（饿死了），它会直接在硅片底层拉低复位引脚，强行把单片机重启！这叫**绝对的物理级保底**。
    

---

### 第二层透视：看门狗的“三板斧”指令

控制这只狗，不需要填复杂的结构体表，只需要往它的核心寄存器（KR，Key Register）里扔三把“钥匙”（特殊的十六进制密码）：

1. **开锁（0x5555）：** 狗默认是被锁在笼子里的，不让你改它的倒计时时间。必须先扔这把钥匙，解除写保护。
    
2. **启动（0xCCCC）：** 扔出这把钥匙，看门狗正式激活，开始疯狂倒计时。一旦启动，**绝不能中途关闭**（除非断电）。
    
3. **喂狗（0xAAAA）：** 这是最重要的动作！在倒计时变成 0 之前，你必须扔出这把钥匙。看门狗一拿到这把钥匙，倒计时就会瞬间恢复到满值（比如重新回到 2 秒），这叫“喂狗”。
    

---

### 第三层实战：如何用看门狗实现“断线保护”？

我们要在这套系统里，建立一套经典的**“心跳包 (Heartbeat)”**机制。

**1. 规则制定：** 规定你的电脑（上位机）只要连着线，除了发 'A', 'B', 'S' 控制电机外，**必须每隔 1 秒钟，向单片机发送一个字母 `'H'`（Heartbeat）**。

**2. 秘书（中断）的任务：** 只要收到字母 `'H'`，秘书什么都不用报告给老板，直接去后院**“喂狗一次 (写入 0xAAAA)”**。

**3. 灾难发生时的自动拯救：** 假设狗的饿死时间被我们设定为 **2 秒**。

- **正常情况：** 电脑每 1 秒发一次 `'H'`，秘书每 1 秒喂一次狗，狗永远饿不死，机器正常运转。
    
- **断线情况：** 杜邦线被踢断，秘书再也收不到 `'H'`。
    
- **1秒过去... 2秒过去...** 狗饿死了！
    
- **狂暴咬人：** 看门狗瞬间触发系统硬件复位（重启）。STM32 重启后，所有的引脚电平归零，电机代码初始化。由于我们写在 `main` 函数开头的是 `Motor_SetState(0);`，**电机瞬间安全锁死停车！**
    

这就是工业界无人值守机器人的标准保命逻辑！

### 第四层代码：把它塞进你的工程里

我们只需要写一个极简的初始化函数，并改造一下你的秘书（串口中断）。

**1. 增加看门狗初始化函数（把这个加在你的代码里）：**

```
void IWDG_Init(void)
{
    // 1. 解除写保护 (扔出 0x5555 钥匙)
    IWDG_WriteAccessCmd(IWDG_WriteAccess_Enable);

    // 2. 设定倒计时速度 (分频比)
    // LSI 内部时钟是 40kHz。除以 64 后，倒计时频率变为 625Hz (每秒数 625 下)
    IWDG_SetPrescaler(IWDG_Prescaler_64);

    // 3. 设定饿死时间 (重装载值)
    // 我们想让它 2 秒钟饿死。2秒 * 625下/秒 = 1250 下。
    IWDG_SetReload(1250);

    // 4. 喂一次狗，让倒计时从 1250 开始 (扔出 0xAAAA 钥匙)
    IWDG_ReloadCounter();

    // 5. 正式放狗！倒计时开始，生死时速启动 (扔出 0xCCCC 钥匙)
    IWDG_Enable();
}
```

**2. 改造秘书（在主函数 `main` 前提下，修改原来的中断服务函数）：**

```
void USART1_IRQHandler(void)
{
    if(USART_GetITStatus(USART1, USART_IT_RXNE) != RESET)
    {
        uint8_t temp_data = USART_ReceiveData(USART1);
        
        // 【新增看门狗逻辑】
        if(temp_data == 'H') 
        {
            // 收到心跳包！秘书立刻去后院喂狗
            IWDG_ReloadCounter(); 
        }
        else 
        {
            // 如果不是心跳包，那就是控制指令，放在桌上给老板看
            g_ReceiveCmd = temp_data;
        }
        
        USART_ClearITPendingBit(USART1, USART_IT_RXNE);
    }
}
```

**3. 在 `main` 函数里挂号：**

```
int main(void)
{
    Motor_Init();               
    UART1_Init_With_Interrupt();
     Motor_SetState(0);

    // 【最后一步：放狗！】
    // 注意：放狗必须放在所有硬件初始化完成之后。
    IWDG_Init();                

   
    while(1)
    {
        // ... 原来的指令解析代码保持完全不变 ...
    }
}
```

把这段代码烧进去，你可以亲自导演一场“通信灾难”：

1. 用电脑串口助手连接，疯狂发送字母 `'H'`（可以勾选串口助手的“定时发送”，设为每 1000ms 发送一次 `'H'`）。
    
2. 发送一个 `'A'`，电机欢快地转动起来。
    
3. **此时，暴力拔掉你接在 PA10 上的那根杜邦线！**
    
4. 倒数 2 秒钟：2... 1...
    
5. **“咔”！** 你会看到 STM32 开发板上的电源灯闪烁了一下（或者电机瞬间停转并发出锁死的低频电流声）。系统成功重启并紧急刹车了！

## 最终代码
```c
#include "stm32f10x.h"

// ==========================================
// 【全局司令部】
// volatile 确保每次查询都读取最新内存，防止被编译器优化
// ==========================================
volatile uint8_t g_ReceiveCmd = 0; 

// ==========================================
// 【API 函数声明】
// ==========================================
void Delay_ms(uint32_t ms);
void Motor_Init(void);
void Motor_SetState(uint8_t state);
void Motor_SetDir(uint8_t dir);
void Motor_SetSpeed_ARR(uint16_t arr_value);
void UART1_Init_With_Interrupt(void); 
void IWDG_Init_Config(void);

// ==========================================
// 【总裁办：主函数】
// 负责业务逻辑编排，永远不阻塞等待
// ==========================================
int main(void)
{
    // 1. 各大底层部门初始化
    Motor_Init();               // 成立电机控制部 (PA0发脉冲, PA1管方向)
    UART1_Init_With_Interrupt();// 成立通信联络部 (PA9/PA10, 开启接收中断)
    
    // 2. 默认开机状态：电机锁死待命
    Motor_SetState(0);

    // 3. 【终极安全锁】放狗！(必须在所有硬件初始化之后执行)
    IWDG_Init_Config();         

    // 4. 异步死循环：老板永远在这里巡视
    while(1)
    {
        // 如果桌上有新指令 (非零)
        if (g_ReceiveCmd != 0) 
        {
            switch (g_ReceiveCmd)
            {
                case 'A': // 【前行指令】
                    Motor_SetDir(1);
                    Motor_SetSpeed_ARR(1200); // 绝对安全起步速度
                    Motor_SetState(1);        // 启动脉冲
                    // 柔性加速到目标速度 (ARR=99，约 60 转/分)
                    for(int arr = 1200; arr >= 99; arr -= 3) {
                        Motor_SetSpeed_ARR(arr); 
                        Delay_ms(3);
                    }
                    break;

                case 'B': // 【倒车指令】
                    Motor_SetDir(0);
                    Motor_SetSpeed_ARR(1200); 
                    Motor_SetState(1);
                    // 柔性加速
                    for(int arr = 1200; arr >= 99; arr -= 3) {
                        Motor_SetSpeed_ARR(arr); 
                        Delay_ms(3);
                    }
                    break;

                case 'S': // 【平稳刹车指令】
                    // 从当前高速平滑降回安全低速
                    for(int arr = 99; arr <= 1200; arr += 3) {
                        Motor_SetSpeed_ARR(arr); 
                        Delay_ms(3);
                    }
                    Motor_SetState(0); // 彻底锁死停机
                    break;
            }
            
            // 指令执行完毕，清空桌面，准备迎接下一条指令
            g_ReceiveCmd = 0; 
        }
        
        // 当没有指令时，CPU 以极快的速度空转。
        // 底层的定时器正在自动发脉冲，保持电机运转，互不干扰！
    }
}

// ==========================================
// 【中断秘书办：串口接收服务函数】
// 专管收信，雷厉风行，绝不拖泥带水
// ==========================================
void USART1_IRQHandler(void)
{
    // 确认确实是“收到数据”触发的警报
    if(USART_GetITStatus(USART1, USART_IT_RXNE) != RESET)
    {
        // 瞬间拿走数据
        uint8_t temp_data = USART_ReceiveData(USART1);
        
        // 核心救命逻辑：拦截心跳包
        if(temp_data == 'H') 
        {
            // 如果是心跳包，直接去后院喂狗，不惊动老板
            IWDG_ReloadCounter(); 
        }
        else 
        {
            // 如果是业务指令 (A/B/S)，放在老板桌上
            g_ReceiveCmd = temp_data;
        }
        
        // 清除警报标志位，准备收下一个字符
        USART_ClearITPendingBit(USART1, USART_IT_RXNE);
    }
}

// ==========================================
// 【部门建设：看门狗初始化】
// ==========================================
void IWDG_Init_Config(void)
{
    IWDG_WriteAccessCmd(IWDG_WriteAccess_Enable); // 1. 开锁
    IWDG_SetPrescaler(IWDG_Prescaler_64);         // 2. 倒计时频率设定为 625Hz
    IWDG_SetReload(1250);                         // 3. 设定 2 秒饿死 (1250 / 625 = 2s)
    IWDG_ReloadCounter();                         // 4. 喂第一次狗，装满倒计时
    IWDG_Enable();                                // 5. 正式放狗，不可撤销！
}

// ==========================================
// 【部门建设：串口与中断初始化】
// ==========================================
void UART1_Init_With_Interrupt(void)
{
    GPIO_InitTypeDef GPIO_InitStructure;
    USART_InitTypeDef USART_InitStructure;
    NVIC_InitTypeDef NVIC_InitStructure;

    RCC_APB2PeriphClockCmd(RCC_APB2Periph_USART1 | RCC_APB2Periph_GPIOA, ENABLE);

    // PA9 (TX 发送)：复用推挽输出
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_9;
    GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_AF_PP;
    GPIO_Init(GPIOA, &GPIO_InitStructure);

    // PA10 (RX 接收)：浮空输入
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_10;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_IN_FLOATING;
    GPIO_Init(GPIOA, &GPIO_InitStructure);

    // 串口配置: 115200 波特率, 8数据位, 1停止位, 无校验
    USART_InitStructure.USART_BaudRate = 115200; 
    USART_InitStructure.USART_WordLength = USART_WordLength_8b; 
    USART_InitStructure.USART_StopBits = USART_StopBits_1;      
    USART_InitStructure.USART_Parity = USART_Parity_No;         
    USART_InitStructure.USART_HardwareFlowControl = USART_HardwareFlowControl_None;
    USART_InitStructure.USART_Mode = USART_Mode_Rx | USART_Mode_Tx; 
    USART_Init(USART1, &USART_InitStructure);

    // 开启接收中断
    USART_ITConfig(USART1, USART_IT_RXNE, ENABLE);

    // NVIC 中断优先级配置
    NVIC_PriorityGroupConfig(NVIC_PriorityGroup_2); 
    NVIC_InitStructure.NVIC_IRQChannel = USART1_IRQn; 
    NVIC_InitStructure.NVIC_IRQChannelPreemptionPriority = 1; // 抢占优先级最高
    NVIC_InitStructure.NVIC_IRQChannelSubPriority = 1;        // 响应优先级最高
    NVIC_InitStructure.NVIC_IRQChannelCmd = ENABLE;           
    NVIC_Init(&NVIC_InitStructure);

    // 合上串口总电闸
    USART_Cmd(USART1, ENABLE);
}

// ==========================================
// 【部门建设：电机驱动初始化与控制API】
// ==========================================
void Motor_Init(void) {
    GPIO_InitTypeDef GPIO_InitStructure;
    TIM_TimeBaseInitTypeDef  TIM_TimeBaseStructure;
    TIM_OCInitTypeDef  TIM_OCInitStructure;

    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA | RCC_APB2Periph_AFIO, ENABLE);
    RCC_APB1PeriphClockCmd(RCC_APB1Periph_TIM2, ENABLE);

    // PA0 (PWM 发脉冲)：复用开漏
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_0;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_AF_OD;
    GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_Init(GPIOA, &GPIO_InitStructure);

    // PA1 (DIR 方向控制)：通用开漏
    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_1;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_Out_OD;
    GPIO_Init(GPIOA, &GPIO_InitStructure);
    GPIO_SetBits(GPIOA, GPIO_Pin_1); 

    // TIM2 时基配置 (2.4MHz 计数频率)
    TIM_TimeBaseStructure.TIM_Prescaler = 30 - 1;     
    TIM_TimeBaseStructure.TIM_Period = 100 - 1; 
    TIM_TimeBaseStructure.TIM_ClockDivision = 0;
    TIM_TimeBaseStructure.TIM_CounterMode = TIM_CounterMode_Up;
    TIM_TimeBaseInit(TIM2, &TIM_TimeBaseStructure);

    // TIM2 PWM 配置 (保持 50% 占空比)
    TIM_OCInitStructure.TIM_OCMode = TIM_OCMode_PWM1;
    TIM_OCInitStructure.TIM_OutputState = TIM_OutputState_Enable;
    TIM_OCInitStructure.TIM_Pulse = 50; 
    TIM_OCInitStructure.TIM_OCPolarity = TIM_OCPolarity_High;
    TIM_OC1Init(TIM2, &TIM_OCInitStructure);

    TIM_OC1PreloadConfig(TIM2, TIM_OCPreload_Enable);
    TIM_ARRPreloadConfig(TIM2, ENABLE);
}

void Motor_SetState(uint8_t state) {
    if(state == 1) TIM_Cmd(TIM2, ENABLE);
    else TIM_Cmd(TIM2, DISABLE);
}

void Motor_SetDir(uint8_t dir) {
    Motor_SetState(0); // 绝对铁律：换向必须先停车
    Delay_ms(2);       
    if(dir == 1) GPIO_SetBits(GPIOA, GPIO_Pin_1);
    else GPIO_ResetBits(GPIOA, GPIO_Pin_1);
}

void Motor_SetSpeed_ARR(uint16_t arr_value) {
    TIM_SetAutoreload(TIM2, arr_value);       
    TIM_SetCompare1(TIM2, arr_value / 2);     
}

// 粗略毫秒级延时
void Delay_ms(uint32_t ms) {
    uint32_t i, j;
    for(i = 0; i < ms; i++)
        for(j = 0; j < 7200; j++); 
}
```
