### 核心物理比喻：老板、邮局与“自动传送带”

**过去的悲剧（CPU 轮询发快递）：** 还记得咱们那段致命的 `while(USART_GetFlagStatus...)` 代码吗？ 这就好比老板（CPU）亲自抱着 8 个包裹去邮局。邮局的窗口一次只能塞一个包裹，而且办事员动作极慢（波特率限制，发一个要 86 微秒）。 老板塞进去一个，**必须傻站在原地死等 86 微秒**，等办事员处理完了，才能塞第二个。 发完 8 个包裹，老板足足在窗口罚站了 0.6 毫秒！这 0.6 毫秒里，如果突发了紧急的中断指令，老板的响应就会变迟钝。

**DMA 的降维打击（全自动传送带）：**

DMA 相当于老板花钱在公司仓库（内存 RAM）和邮局窗口（串口 TX 寄存器）之间，架设了一条**完全独立于老板的“物理传送带”**。 现在老板发快递是这样的：

1. 老板把 8 个包裹按顺序放在仓库的地上（给数组赋值）。
    
2. 老板按下传送带的启动电钮，大喊一声：“传送带，把这 8 个包裹发出去！”
    
3. **老板瞬间转身走人，耗时 1 微秒！回去接着跑 `while(1)` 状态机！**
    
4. 传送带（DMA 硬件）会在后台，默默地盯着邮局窗口。只要窗口一空，传送带就自动抓起地上的一个包裹扔进去，直到 8 个全部扔完自动停机。**全程不需要老板（CPU）插手！**

## 主体代码
```c
#include "stm32f10x.h"

// ==========================================
// 【全局时钟】
// ==========================================
volatile uint32_t g_SystemTime = 0;      

// ==========================================
// 【通讯接收部门：收快递专用变量】
// ==========================================
volatile uint8_t  g_NewFrameReady = 0;   
volatile uint8_t  g_ParsedCmd = 0;       
volatile uint16_t g_ParsedARR = 0;       

// ==========================================
// 【电机部门：状态机专用变量】
// ==========================================
uint8_t  g_MotorState   = 0;             
uint16_t g_CurrentARR   = 1200;          
uint16_t g_TargetARR    = 1200;          
uint32_t g_LastStepTime = 0;             

// ==========================================
// 【通讯发送部门：播报员与 DMA 传送带专用】
// ==========================================
uint32_t g_LastReportTime = 0;           // 播报员的秒表
uint8_t  g_ReportPacket[8];              // 【划重点】存放在物理内存里的 DMA 专属固定仓库！

// ==========================================
// 【底层 API 声明】
// ==========================================
void Motor_Init(void);
void Motor_SetState(uint8_t state);
void Motor_SetDir(uint8_t dir);
void Motor_SetSpeed_ARR(uint16_t arr_value);
void UART1_Init_With_Interrupt(void); 
void UART1_TX_DMA_Init(void);            // 【新增】安装 DMA 传送带
void IWDG_Init_Config(void);
void Send_Status_Report(void);           // 【修改】光速打包函数

// ==========================================
// 【心脏起搏器】
// ==========================================
void SysTick_Handler(void) 
{
    g_SystemTime++; 
}

// ==========================================
// 【总裁办：主函数】
// ==========================================
int main(void)
{
    // 1. 各大部门成立
    Motor_Init();               
    UART1_Init_With_Interrupt();
    UART1_TX_DMA_Init();        // 【新增】开启 DMA 自动传送带！
    Motor_SetState(0);
    
    // 2. 启动系统时钟与保命看门狗
    SysTick_Config(SystemCoreClock / 1000); 
    IWDG_Init_Config();         

    // 3. 极速轮询跑道
    while(1)
    {
        // ---------------------------------------------------
        // 任务一：极速指令响应 (处理外来命令)
        // ---------------------------------------------------
        if (g_NewFrameReady == 1) 
        {
            switch (g_ParsedCmd)
            {
                case 0x01: 
                    if(g_MotorState == 0) {   
                        Motor_SetDir(1);
                        g_CurrentARR = 1200;  
                        g_TargetARR = g_ParsedARR; 
                        Motor_SetSpeed_ARR(g_CurrentARR);
                        Motor_SetState(1);    
                        g_MotorState = 1;     
                        g_LastStepTime = g_SystemTime; 
                    }
                    else if(g_MotorState == 2 || g_MotorState == 1 || g_MotorState == 3) {
                        g_TargetARR = g_ParsedARR;
                        if(g_CurrentARR > g_TargetARR) g_MotorState = 1; 
                        else if(g_CurrentARR < g_TargetARR) g_MotorState = 3; 
                    }
                    break;
                case 0x02: 
                    if(g_MotorState == 0) {
                        Motor_SetDir(0);
                        g_CurrentARR = 1200;  
                        g_TargetARR = g_ParsedARR; 
                        Motor_SetSpeed_ARR(g_CurrentARR);
                        Motor_SetState(1);    
                        g_MotorState = 1;     
                        g_LastStepTime = g_SystemTime; 
                    }
                    break;
                case 0x03: 
                    g_TargetARR = 1200;       
                    g_MotorState = 3;         
                    break;
            }
            g_NewFrameReady = 0; 
        }

        // ---------------------------------------------------
        // 任务二：电机平滑调速状态机 (微操执行)
        // ---------------------------------------------------
        if (g_MotorState == 1) 
        {
            if (g_SystemTime - g_LastStepTime >= 3) 
            {
                g_LastStepTime = g_SystemTime; 
                g_CurrentARR -= 3;             
                Motor_SetSpeed_ARR(g_CurrentARR);
                if (g_CurrentARR <= g_TargetARR) g_MotorState = 2;          
            }
        }
        else if (g_MotorState == 3)
        {
            if (g_SystemTime - g_LastStepTime >= 3) 
            {
                g_LastStepTime = g_SystemTime; 
                g_CurrentARR += 3;             
                Motor_SetSpeed_ARR(g_CurrentARR);
                if (g_CurrentARR >= g_TargetARR) {
                    if (g_TargetARR == 1200) { 
                        Motor_SetState(0); 
                        g_MotorState = 0;  
                    } else {
                        g_MotorState = 2;  
                    }
                }
            }
        }

        // ---------------------------------------------------
        // 任务三：定时发送健康报告 (向电脑汇报)
        // ---------------------------------------------------
        if (g_SystemTime - g_LastReportTime >= 100) 
        {
            g_LastReportTime = g_SystemTime; // 刷新秒表
            Send_Status_Report();            // 光速打包并启动 DMA，绝不死等！
        }
    }
}

// ==========================================
// 【新增极速功能：DMA 底层发快递】
// CPU 耗时几乎为 0，不阻塞任何状态机！
// ==========================================
void Send_Status_Report(void) 
{
    // 1. 老板冲进仓库，以光速更新固定货位上的 8 个包裹
    g_ReportPacket[0] = 0xAA;
    g_ReportPacket[1] = 0x55;
    g_ReportPacket[2] = 0x81;                      
    g_ReportPacket[3] = g_MotorState;              
    g_ReportPacket[4] = (g_CurrentARR >> 8) & 0xFF;
    g_ReportPacket[5] = g_CurrentARR & 0xFF;       
    g_ReportPacket[6] = (uint8_t)(g_ReportPacket[2] + g_ReportPacket[3] + g_ReportPacket[4] + g_ReportPacket[5]);
    g_ReportPacket[7] = 0xFF;

    // 2. 按下 DMA 传送带的启动电钮！
    DMA_Cmd(DMA1_Channel4, DISABLE);           // 先拉闸停机
    DMA_SetCurrDataCounter(DMA1_Channel4, 8);  // 告诉传送带：数量重新恢复为 8 个！
    DMA_Cmd(DMA1_Channel4, ENABLE);            // 合闸启动！传送带在后台静默接管发送任务！
    
    // 3. 老板瞬间转身离开！再也没有 while 死等了！
}

// ==========================================
// 【中断秘书办：包裹分拣状态机】
// ==========================================
void USART1_IRQHandler(void)
{
    static uint8_t rx_state = 0;     
    static uint8_t rx_cmd = 0;       
    static uint8_t rx_arr_h = 0;     
    static uint8_t rx_arr_l = 0;     
    static uint8_t rx_checksum = 0;  

    if(USART_GetITStatus(USART1, USART_IT_RXNE) != RESET)
    {
        uint8_t rx_data = USART_ReceiveData(USART1); 
        
        switch(rx_state) 
        {
            case 0: 
                if(rx_data == 0xAA) rx_state = 1; else rx_state = 0; break;
            case 1: 
                if(rx_data == 0x55) rx_state = 2; else if(rx_data == 0xAA) rx_state = 1; else rx_state = 0; break;
            case 2: 
                rx_cmd = rx_data; rx_state = 3; break;
            case 3: 
                rx_arr_h = rx_data; rx_state = 4; break;
            case 4: 
                rx_arr_l = rx_data; rx_state = 5; break;
            case 5: 
                rx_checksum = rx_data; rx_state = 6; break;
            case 6: 
                if(rx_data == 0xFF) 
                {
                    uint8_t calc_sum = (uint8_t)(rx_cmd + rx_arr_h + rx_arr_l);
                    if(calc_sum == rx_checksum) 
                    {
                        if(rx_cmd == 0x00) {
                            IWDG_ReloadCounter();
                        } else {
                            g_ParsedCmd = rx_cmd;
                            g_ParsedARR = (rx_arr_h << 8) | rx_arr_l; 
                            g_NewFrameReady = 1; 
                            IWDG_ReloadCounter();
                        }
                    }
                }
                rx_state = 0; break;
        }
        USART_ClearITPendingBit(USART1, USART_IT_RXNE);
    }
}

// ==========================================
// 【基础建设部门：安装 DMA 传送带】
// ==========================================
void UART1_TX_DMA_Init(void) 
{
    DMA_InitTypeDef DMA_InitStructure;
    
    // 给传送带通电 (打开 DMA1 时钟)
    RCC_AHBPeriphClockCmd(RCC_AHBPeriph_DMA1, ENABLE);

    // 拆除旧传送带，准备安装新的 (复位 DMA1 通道4)
    DMA_DeInit(DMA1_Channel4);
    
    // 配置传送带的终点：串口1 的邮局窗口 (DR寄存器)
    DMA_InitStructure.DMA_PeripheralBaseAddr = (uint32_t)&USART1->DR; 
    // 配置传送带的起点：咱们的仓库货位 (g_ReportPacket)
    DMA_InitStructure.DMA_MemoryBaseAddr = (uint32_t)g_ReportPacket;  
    // 设定运输方向：从 内存 运往 外设
    DMA_InitStructure.DMA_DIR = DMA_DIR_PeripheralDST;                
    // 每次运输的数量：8 个包裹
    DMA_InitStructure.DMA_BufferSize = 8;                             
    // 邮局窗口不动，所以外设地址不递增
    DMA_InitStructure.DMA_PeripheralInc = DMA_PeripheralInc_Disable;  
    // 仓库包裹是排成一排的，每次抓完内存地址要往后挪一格
    DMA_InitStructure.DMA_MemoryInc = DMA_MemoryInc_Enable;           
    // 包裹大小：字节 (8位)
    DMA_InitStructure.DMA_PeripheralDataSize = DMA_PeripheralDataSize_Byte;
    DMA_InitStructure.DMA_MemoryDataSize = DMA_MemoryDataSize_Byte;
    // 模式：运完 8 个就自动停机 (Normal)
    DMA_InitStructure.DMA_Mode = DMA_Mode_Normal;                     
    DMA_InitStructure.DMA_Priority = DMA_Priority_High;
    DMA_InitStructure.DMA_M2M = DMA_M2M_Disable;
    
    DMA_Init(DMA1_Channel4, &DMA_InitStructure);

    // 【致命一击】告诉串口1：发货任务移交给 DMA，别烦 CPU！
    USART_DMACmd(USART1, USART_DMAReq_Tx, ENABLE);
}

void Motor_Init(void) {
    GPIO_InitTypeDef GPIO_InitStructure;
    TIM_TimeBaseInitTypeDef  TIM_TimeBaseStructure;
    TIM_OCInitTypeDef  TIM_OCInitStructure;

    RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOA | RCC_APB2Periph_AFIO, ENABLE);
    RCC_APB1PeriphClockCmd(RCC_APB1Periph_TIM2, ENABLE);

    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_0;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_AF_OD;
    GPIO_InitStructure.GPIO_Speed = GPIO_Speed_50MHz;
    GPIO_Init(GPIOA, &GPIO_InitStructure);

    GPIO_InitStructure.GPIO_Pin = GPIO_Pin_1;
    GPIO_InitStructure.GPIO_Mode = GPIO_Mode_Out_OD;
    GPIO_Init(GPIOA, &GPIO_InitStructure);
    GPIO_SetBits(GPIOA, GPIO_Pin_1); 

    TIM_TimeBaseStructure.TIM_Prescaler = 30 - 1;     
    TIM_TimeBaseStructure.TIM_Period = 100 - 1; 
    TIM_TimeBaseStructure.TIM_ClockDivision = 0;
    TIM_TimeBaseStructure.TIM_CounterMode = TIM_CounterMode_Up;
    TIM_TimeBaseInit(TIM2, &TIM_TimeBaseStructure);

    TIM_OCInitStructure.TIM_OCMode = TIM_OCMode_PWM1;
    TIM_OCInitStructure.TIM_OutputState = TIM_OutputState_Enable;
    TIM_OCInitStructure.TIM_Pulse = 50; 
    TIM_OCInitStructure.TIM_OCPolarity = TIM_OCPolarity_High;
    TIM_OC1Init(TIM2, &TIM_OCInitStructure);

    TIM_OC1PreloadConfig(TIM2, TIM_OCPreload_Enable);
    TIM_ARRPreloadConfig(TIM2, ENABLE);
}

void Motor_SetState(uint8_t state) {
    if(state == 1) TIM_Cmd(TIM2, ENABLE);
    else TIM_Cmd(TIM2, DISABLE);
}

void Motor_SetDir(uint8_t dir) {
    Motor_SetState(0); 
    uint32_t start_time = g_SystemTime;
    while(g_SystemTime - start_time < 2); 
    if(dir == 1) GPIO_SetBits(GPIOA, GPIO_Pin_1);
    else GPIO_ResetBits(GPIOA, GPIO_Pin_1);
}

void Motor_SetSpeed_ARR(uint16_t arr_value) {
    TIM_SetAutoreload(TIM2, arr_value);       
    TIM_SetCompare1(TIM2, arr_value / 2);     
}
```

## 新增功能代码答疑
### 第一块：申请一块“永久物理地皮” (全局变量)

```
uint8_t  g_ReportPacket[8]; 
```

- **大白话：** 为什么要把它变成全局变量？ 如果是以前那种局部变量（写在函数内部的 `uint8_t report_packet[8]`），当函数运行结束时，这块临时内存（栈内存）就会被系统立刻回收、销毁。 但是现在用了 DMA，**老板（CPU）只要一按启动键就会离开函数。** 如果包裹放在临时内存里，等传送带慢吞吞去拿的时候，包裹早就凭空消失了！传送带会抓取一堆乱码发出去。 所以，必须用全局变量，在 RAM 里划出一块“永久存在的仓库地皮”，专门给传送带放包裹。
    

---
### 第二块：传送带工程图纸 ( `UART1_TX_DMA_Init` 函数 )

这是全篇最硬核的地方，咱们一行行“拧螺丝”：
```
void UART1_TX_DMA_Init(void) 
{
    DMA_InitTypeDef DMA_InitStructure;
    
    // 1. 给传送带拉专线供电
    RCC_AHBPeriphClockCmd(RCC_AHBPeriph_DMA1, ENABLE);
```

- **大白话：** STM32 为了省电，所有硬件默认都是断电的。`RCC_AHB...` 就是给 DMA1 部门通电。注意看，它是 **AHB**（高速总线），比咱们之前串口用的 APB 还要高级，说明 DMA 是单片机里的“特权阶级”。
    
```
    // 2. 拆除违章建筑，恢复出厂设置
    DMA_DeInit(DMA1_Channel4);
```

- **大白话：** 万一之前别的程序瞎搞过这条传送带，咱们先把它复原。**为什么是 Channel4（通道 4）？** 这是 STM32 芯片出厂时在硅片上焊死的物理连线：串口 1 的发送端口，必须匹配 DMA1 的通道 4。没得选，只能查手册背下来。
    
```
    // 3. 设定收货地址（邮局窗口）
    DMA_InitStructure.DMA_PeripheralBaseAddr = (uint32_t)&USART1->DR; 
```

- **大白话：** `Peripheral` 意思是外设（串口）。`USART1->DR` 就是串口内部那个用来发送数据的寄存器（也就是邮局窗口）。前面的 `&` 代表“取地址”。这行代码告诉传送带：**你的终点站就在这个物理坐标。**
    
```
    // 4. 设定发货地址（仓库地上）
    DMA_InitStructure.DMA_MemoryBaseAddr = (uint32_t)g_ReportPacket;  
```

- **大白话：** `Memory` 意思是内存。把咱们刚才申请的那块永久地皮 `g_ReportPacket` 的首地址告诉传送带：**你的起点站在这里。**
    
```
    // 5. 设定运输方向
    DMA_InitStructure.DMA_DIR = DMA_DIR_PeripheralDST;                
```

- **大白话：** `DST` 是 Destination（目的地）的缩写。意思是：以外设为目的地。方向是 **内存 -> 外设**。
    
```
    // 6. 设定卡车容量
    DMA_InitStructure.DMA_BufferSize = 8;                             
```

- **大白话：** 明确告诉传送带，这趟活儿总共有 8 个包裹要运。它内部会有一个倒数计数器，发一个减一个，减到 0 就停。
    
```
    // 7. 【极致物理法则】收货地址（邮局窗口）要不要移动？
    DMA_InitStructure.DMA_PeripheralInc = DMA_PeripheralInc_Disable;  
```

- **大白话：** `Inc` 是 Increment（递增）。串口的发送窗口永远只有一个洞（`DR` 寄存器地址是固定的）。传送带每次都必须往同一个洞里塞，所以**禁止（Disable）**外设地址递增。
    
```
    // 8. 【极致物理法则】发货地址（仓库地上）要不要移动？
    DMA_InitStructure.DMA_MemoryInc = DMA_MemoryInc_Enable;           
```

- **大白话：** 咱们仓库里的 8 个包裹是排成一排放在地上的（数组的 8 个元素）。传送带抓完第 0 个包裹，机械臂必须**往旁边移动一格**，去抓第 1 个包裹，不然它就一直死拽着第 0 个包裹发。所以**允许（Enable）**内存地址递增。
    
```
    // 9. 设定包裹的尺寸
    DMA_InitStructure.DMA_PeripheralDataSize = DMA_PeripheralDataSize_Byte;
    DMA_InitStructure.DMA_MemoryDataSize = DMA_MemoryDataSize_Byte;
```

- **大白话：** 告诉传送带，每次抓取和塞入的尺寸都是 `Byte`（1个字节，8位）。别搞错了抓成 `HalfWord`（半字，16位）。
    
```
    // 10. 设定工作模式
    DMA_InitStructure.DMA_Mode = DMA_Mode_Normal;                     
```

- **大白话：** `Normal` 正常模式。意思是发完这 8 个包裹，传送带就原地关机睡觉，等老板下次再叫。 _(扩展：如果是 `Circular` 循环模式，发完 8 个它会瞬间倒带回第 0 个，永远发下去，像复读机一样，通常给屏幕刷图用。)_
    
```
    // (优先级和内存到内存模式，这里先不管，照抄即可)
    DMA_InitStructure.DMA_Priority = DMA_Priority_High;
    DMA_InitStructure.DMA_M2M = DMA_M2M_Disable;
    
    // 11. 按照上面的图纸，把传送带正式建好！
    DMA_Init(DMA1_Channel4, &DMA_InitStructure);

    // 12. 【全场最致命的连线】
    USART_DMACmd(USART1, USART_DMAReq_Tx, ENABLE);
}
```

- **大白话：** 最后这一行至关重要。`DMAReq_Tx` 意思是“DMA的发送请求”。这行代码相当于在串口硬件内部拨动了一个开关：**“喂，串口，以后你只要窗口空了，直接给 DMA 传送带打电话叫它送货，千万别拉响中断去烦 CPU 老板了！”**
    

---
### 第三块：老板发号施令 ( `Send_Status_Report` 函数 )

现在的发包裹函数，变得极度清爽。上半部分 `g_ReportPacket[0] = ...` 就是老板在地上摆放包裹并算出校验和，不用多说。

关键是下半部分触发 DMA 的这三行：
```
    DMA_Cmd(DMA1_Channel4, DISABLE);           // 先拉闸停机
    DMA_SetCurrDataCounter(DMA1_Channel4, 8);  // 重新设定数量
    DMA_Cmd(DMA1_Channel4, ENABLE);            // 合闸启动！
```

- **大白话：** STM32 的硬件有一个死规定：**如果 DMA 传送带还在通电状态，你是绝对不能修改它的剩余数量的！** 而且 DMA 处于 `Normal` 模式，发完 8 个包裹后，它内部的计数器变成了 0。你不给它重置成 8，它下次通电也不会干活。 所以，标准操作必须是：
    
    1. **强制拉闸断电** (`DISABLE`)。
        
    2. 把挂在墙上的倒数计数板**拨回 8** (`DMA_SetCurrDataCounter`)。
        
    3. **重新合闸通电** (`ENABLE`)。
        

此时，伴随着虚拟的“轰隆隆”声，传送带开始在后台静默运行。老板（CPU）拍拍手上的灰尘，瞬间转身走出了这个函数，去忙下一件事了。
